# PaperAnalysis
以SSM框架为核心的高校试卷分析系统


### 运行方法
#### 环境
jdk8，Tomcat8.5，MySQL5，Maven3.5


### 系统介绍
#### 总体功能
本系统实现了以下功能：游客生成试卷分析报告、下载简易分析报告、登录、注册、重置密码、用户生成总成绩分析报告、用户生成知识点分析报告、用户管理知识点、用户管理历史报告。
